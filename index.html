LTI Systems

入力、インパルス応答、畳み込み、伝達関数、周波数応答を1つの流れで見ます。

time / frequency
Input x[n]
Impulse response h[n]
Convolution y[n] = x[n] * h[n]
Transfer function H(z)
Frequency response H(ejω)

Impulse Response

システムに単位インパルスを入れたときの出力。LTIではこれだけでシステム全体が決まります。

Time Convolution

入力波形をずらしながら、インパルス応答と重ねて足し合わせます。

Frequency Response

畳み込みは周波数領域では掛け算になります。音色変化を見せる中心です。

FIR/IIR & Stability

有限長の畳み込み、フィードバック、極と零点、安定性をまとめて扱います。

poles / zeros
FIR y[n] = Σ b[k]x[n-k]

過去の入力だけを使う。インパルス応答は有限長で、安定性を扱いやすい。

IIR y[n] = Σ b[k]x[n-k] - Σ a[k]y[n-k]

過去の出力を戻す。少ない係数で強い特性を作れるが、安定性の確認が重要。

Transfer Function

差分方程式をz変換してH(z)で表すと、FIR/IIRの構造が同じ図で見えます。

Stability

離散時間IIRでは極が単位円の内側にあると安定。外に出ると発散します。
